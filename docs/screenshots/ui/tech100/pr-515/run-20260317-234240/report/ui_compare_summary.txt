TECH100 Portfolio refinement review
Run ID: 20260317-234240
PR: #515

Defects found in the live PR #514 page before refinement
- Header clutter from source/date/rebalance pills pushed the actual analytics below the fold.
- Six repeated model cards created visual noise and made comparison harder instead of easier.
- Controls were fragmented into too many small buttons and did not behave like a unified workspace.
- The holdings and attribution presentation felt cramped and forced dense horizontal scanning.
- Methodology and interpretation were buried as a static block instead of attached to the active analysis.
- Chart formatting and data handling had correctness risks: basis-point labels were misleading, comparison series could misalign by date, and the "Max" range silently truncated history.

What changed in this refinement
- Replaced the repeated summary cards with a compact model rail plus a single dynamic workspace summary.
- Consolidated range, benchmark, metric, view, attribution, and holdings controls into one sticky control bar.
- Promoted one larger primary comparison canvas and paired it with cleaner supporting side panels.
- Reworked holdings, attribution, and sector sections around switchable views and ranked summaries.
- Added contextual disclosure panels for interpretation, model meaning, research inputs, and deferred analytics.
- Fixed full-history range loading, date-aligned chart series, basis-point rendering, and percentage/date formatting.
